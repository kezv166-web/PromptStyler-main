# Privacy Policy for PromptStyler

PromptStyler is built as a privacy-first tool. This page informs you of our policies regarding the collection, use, and disclosure of personal data when you use our extension.

## 1. Information Collection and Use
PromptStyler **does not collect, store, or transmit** any personal identification data, browsing history, or usage metrics to external servers. 

## 2. API Keys and Storage
- **API Keys:** Your Groq API key is stored locally in Chrome's secure storage system (`chrome.storage.sync` or `chrome.storage.local`). It is only used to authenticate direct requests to the Groq API.
- **Refinement History & Feedback:** Prompt history and your feedback ratings (thumbs-up/down) are stored locally on your machine in a sandboxed browser database (IndexedDB). This data never leaves your device and is kept entirely local.

## 3. Third-Party Services
The extension sends your prompts and API key directly to **Groq Cloud API** (https://api.groq.com) for processing. No middleman servers are used. Please refer to Groq's privacy policy for how they handle API request data.

## 4. Changes to This Privacy Policy
We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.

## 5. Contact Us
If you have any questions or suggestions about this Privacy Policy, please open an issue on our GitHub repository.
